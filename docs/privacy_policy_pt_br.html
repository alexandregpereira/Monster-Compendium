Política de Privacidade

Última atualização: 11 de maio de 2026

Visão Geral

O Monster Compendium coleta dados de uso limitados e anônimos para melhorar a experiência no aplicativo. Nenhuma informação pessoal identificável (nome, e-mail, localização, etc.) é coletada ou armazenada.

Dados Coletados

Análise de Uso (Amplitude)

Eventos de uso são enviados ao Amplitude para entender como o aplicativo é utilizado. O Amplitude é usado no Android, iOS e desktop. Esses eventos incluem:

  • Consultas de pesquisa e contagem de resultados
  • Interações com monstros e magias
  • Alterações de configurações (ex.: idioma, aparência)
  • Interações com o fluxo de assinatura e paywall
  • Visualizações de banners de anúncios

Os eventos contêm apenas dados de interação com o aplicativo — nenhum identificador pessoal é anexado.

Análise de Uso (Firebase Analytics)

O Firebase Analytics é usado em versões de lançamento do Android para entender padrões de uso. Nenhum dado pessoal é incluído.

Relatório de Erros (Firebase Crashlytics)

No Android e iOS, relatórios de falhas e erros são enviados automaticamente ao Firebase Crashlytics. Esses relatórios incluem modelo do dispositivo, versão do sistema operacional e um rastreamento de pilha. Nenhum dado pessoal é incluído.

Publicidade (Google AdMob)

A versão gratuita do aplicativo exibe banners de anúncios fornecidos pelo Google AdMob no Android e iOS. O AdMob pode usar identificadores de publicidade do dispositivo e dados de uso para exibir anúncios personalizados, de acordo com a Política de Privacidade do Google. No iOS, uma solicitação de consentimento é exibida no primeiro acesso, conforme o framework App Tracking Transparency (ATT) da Apple.

Assinantes premium não visualizam anúncios.

Compras no Aplicativo (RevenueCat)

As compras de assinatura no Android (Google Play) e iOS (App Store) são processadas pelo RevenueCat. O RevenueCat recebe dados de transações de compra para validar e gerenciar direitos de acesso. Consulte a Política de Privacidade do RevenueCat para mais detalhes.

Compartilhamento de Dados

Os dados são compartilhados apenas com os provedores de serviços terceiros listados acima (Amplitude, Firebase, Google AdMob, RevenueCat) para as finalidades descritas. Nenhum dado é vendido a terceiros.
